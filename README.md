# 📦 Cardboard Genie (纸箱焕新工坊)

> Turn waste into wonder. Describe your cardboard boxes and let AI generate clever crafting ideas and step-by-step guides.

**Cardboard Genie** is a smart DIY companion designed to reduce waste and spark creativity. Instead of throwing away delivery boxes or shoe boxes, simply tell the AI what you have, and get instant, tailored upcycling ideas ranging from home organizers to pet toys.

### ✨ Key Features
- **AI-Driven Creativity:** Get personalized projects based on your specific box type.
- **Step-by-Step Guides:** Detailed instructions generated by Gemini AI.
- **Smart Difficulty Grading:** Projects categorized from "Beginner" to "Master Crafter."
- **Eco-Friendly:** Promote a sustainable lifestyle through creative reuse.

### 🚀 Tech Stack
- **Frontend:** React + TypeScript + Tailwind CSS
- **AI Model:** Google Gemini API (via @google/genai)
- **Icons:** Lucide React

### 🛠️ Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/cardboard-genie.git
   cd cardboard-genie
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up Environment Variables:**
   Create a `.env` file in the root directory and add your Gemini API Key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
